# Before doing any steps,

First, go to https://www.microsoft.com/en-us/sql-server/sql-server-downloads

		-	download "Express" edition.

Then, go to https://learn.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-ver16#download-ssms

	- download SSMS by click on " Download SQL Server Management Studio (SSMS) 20.0" text.
	
	- for dedails of every installation, you can watch on "https://www.youtube.com/@weshare5776".
	
	- If you are a version 1.0.0 user, you just have to run the version 1.1.0 stored procedure and software installation.
